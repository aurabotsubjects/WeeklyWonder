This folder is intentionally empty in the git repo.

GIFs are hosted externally (e.g. Cloudflare R2) to keep this repository
small. See the main README.md for how to set `ASSET_BASE_URL` in
`config.js` to point at your bucket.

If you'd rather keep GIFs in this repo instead of a bucket, just drop
the .gif files back into this folder (same filenames as referenced in
data/wonders-0N.js) and leave ASSET_BASE_URL empty in config.js.
